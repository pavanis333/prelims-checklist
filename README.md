# UPSC Prelims Mastery Checklist (React App)

A premium, interactive React application designed for UPSC aspirants to track their preparation.

## Features

-   **Subject-wise Tracking**: Dedicated sections for History (Indian Culture), Geography, and Polity.
-   **3-Stage Mastery**: Track progress for each topic: **Read**, **Revised**, and **Practiced**.
-   **Visual Analytics**: Beautiful circular progress rings for overall and subject-level completion.
-   **Gamification**: Streak counter and Confetti celebrations.
-   **Premium UI**: Glassmorphism design with a dark, focus-oriented theme.
-   **Local Storage**: Progress is saved automatically.

## Tech Stack

-   **React** (Vite)
-   **React Router** for navigation
-   **Vanilla CSS** (Glassmorphism)

## How to Run

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Start Development Server**:
    ```bash
    npm run dev
    ```
    Open `http://localhost:5173` in your browser.

## Build for Production

```bash
npm run build
npm run preview
```
